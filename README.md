# 🛍️ B-Shop | Next-Gen E-Commerce Website

A modern, stylish, and fully responsive **E-Commerce Website UI** built using **HTML, CSS, and JavaScript**.
This project simulates a real-world online shopping experience with product browsing, analytics dashboard, and interactive UI components.

---

## 🌐 Live Demo

🚀 **Explore Here:**
👉 https://shivank302.github.io/bshop-ecommerce-website-/

---

## ✨ Features

✔️ 🛒 Beautiful Product Listing UI
✔️ 🔍 Smart Search & Category Filter
✔️ 📊 Interactive Sales Analytics (Chart.js)
✔️ 👤 User Profile & Order Pages
✔️ 🧑‍💻 Admin Dashboard Panel
✔️ 📦 Order Confirmation System
✔️ 📱 Fully Responsive Design (Mobile + Desktop)
✔️ 📞 Contact Form Section
